# Startup Funding Success Predictor

ML pipeline predicting Indian startup funding success with E-Cell & KDEM research data.

## Key Results
- XGBoost (GridSearchCV-tuned): **AUC 0.83**
- SHAP beeswarm & waterfall plots for explainability

## Tech Stack
Python · XGBoost · SHAP · Scikit-learn · Streamlit · Pandas

## Run
pip install -r requirements.txt
python startup_predictor.py
